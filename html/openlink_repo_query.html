Get a list of bond, bill, ECP and bond future positions for the EEA

Description

Get a list of bond, bill, ECP and bond future positions for the EEA

Usage

openlink_repo_query(
  portfolios = openlink_portfolio_mappings()$Portfolio,
  ins_types = c("GBND", "MM-G-Bill", "MM-ECP", "BONDFUT"),
  tran_statuses = "Validated"
)

Arguments

portfolios

A vector of portfolio names e.g. "CBA - CAN$ Bond Trading"

ins_types

A vector of instrument types e.g. "GBND"

tran_statuses

A vector of transaction statuses e.g. "Validated"

Value

A data frame
