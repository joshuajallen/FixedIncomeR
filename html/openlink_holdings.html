openlink_holdings {FIRVrBOE}R Documentation

Get Openlink holdings by currency and portfolio class

Description

Get Openlink holdings by currency and portfolio class

Usage

openlink_holdings(
  business = "EEA",
  currency = c("AUD", "CAD", "EUR", "JPY", "USD"),
  target_date = Sys.Date()
)

Arguments

business

Business area e.g. "EEA"

currency

Holding currency e.g. "USD"

target_date

Date for which the holding is valid

Value

A data frame of holdings, settled as of a given date

Examples

openlink_holdings()
