openlink_query_api {FIRVrBOE}R Documentation

Access to the RM View over the Openlink API

Description

Access to the RM View over the Openlink API

Usage

openlink_query_api(
  instrument_types = NULL,
  portfolios = NULL,
  tran_statuses = NULL,
  deal_number_min = NULL,
  deal_number_max = NULL,
  trade_date_min = NULL,
  trade_date_max = NULL
)

Arguments

instrument_types

A vector of instrument types e.g. "GBND", "MM-G-Bill"

portfolios

A vector of portfolio names e.g. "CBA - CAN$ Bond Trading"

tran_statuses

A vector of transaction statuses e.g. "Validated"

deal_number_min

Minimum deal number

deal_number_max

Maximum deal number

trade_date_min

Minimum trade date

trade_date_max

Maximum trade date

Value

A data_frame of trades or a zero row data_frame if an error occurred

Examples

openlink_query_api(portfolios = "CBA - CAN$ Bond Trading",
                             instrument_types = c("GBND"),
                             tran_statuses = "Validated")
